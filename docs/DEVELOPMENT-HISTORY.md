# Development history

AgentWorkbench Arena was created during the OpenAI Build Week contest period on July 19, 2026. The merged implementation history through July 21 contains 75 commits and eight reviewed pull requests.

## Milestones

- PR #1 repaired the native multi-candidate runner, worktree isolation, evidence capture, and Windows behavior.
- PR #2 added deterministic telemetry, validation, change analysis, evidence completeness, and hard gates.
- PR #3 added finalized-packet validation, identity masking, read-only GPT-5.6 Sol adjudication, and controller-owned evaluation.
- PR #4 added deterministic static HTML and non-operative YAML reporting plus the sanitized fallback demo.
- PR #5 added the complete workflow, safe templates, preview/topology, decision lenses, coverage, verification, and scheduler-trial preparation.
- PR #6 froze the scheduler baseline contract and added route doctoring, OpenCode configuration layering, clean-clone verification, and sample sanitation.
- PR #7 stabilized exact-byte diagnostics; all six intended flagship configurations passed the final shared diagnostic contract once, with Windows CI green. The flagship comparison itself was not run.
- PR #8 improved judge-facing report presentation and live onboarding without changing evidence or decision authority.

Important repairs included process-tree termination, transport-only retry hygiene, unknown-versus-zero telemetry, failed-over-unavailable gate precedence, dependency-state analysis, task-contract integrity, identity/path leakage protection, isolated judge staging, bounded packets/responses, one-repair enforcement, report immutability, and public-sample sanitation.

The human owned the product, fairness, authority, scope, and release decisions. Codex implemented and tested the repository changes. GPT-5.6 Sol participates inside Arena only as the identity-masked semantic adjudicator; deterministic eligibility remains authoritative.

Current status is maintained in [submission readiness](SUBMISSION-READINESS.md), not in historical phase diaries.
